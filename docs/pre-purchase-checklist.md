# Pre-Purchase Checklist: AC System Replacement

Use this checklist before signing any contract. Print it out and bring it to contractor appointments.

## Before Getting Quotes

- [ ] Document your current system: brand, model, tonnage, SEER rating, installation date
- [ ] Note current comfort issues: hot rooms, humidity problems, uneven cooling, noise
- [ ] Check your FPL bill for average monthly usage (summer vs. winter)
- [ ] Photograph your indoor and outdoor units, thermostat, and ductwork access points
- [ ] List any home modifications since last AC install (additions, new windows, insulation changes)
- [ ] Research current FPL rebates and federal tax credits

## During Contractor Visit

- [ ] Contractor performs Manual J load calculation (not square footage estimate)
- [ ] Contractor inspects existing ductwork condition
- [ ] Contractor checks electrical panel capacity for new equipment
- [ ] Contractor measures refrigerant line distances
- [ ] Contractor assesses drainage options
- [ ] Ask about R-454B vs R-410A options
- [ ] Ask about variable speed vs single/two-stage recommendations
- [ ] Request specific equipment model numbers (not just brand)

## After Installation

- [ ] Register warranty with manufacturer within 60 days
- [ ] Verify permit was pulled and inspection scheduled
- [ ] Test every room for airflow and temperature
- [ ] Check humidity levels (target: 45-55%)
- [ ] Save all paperwork: invoice, warranty registration, permit, model numbers
- [ ] Set up maintenance schedule (twice per year minimum)
- [ ] Program thermostat for optimal schedule
- [ ] Change filter reminder: every 30-90 days depending on MERV rating

## Emergency Reference

If your new system develops issues:
- **No cooling**: Check thermostat batteries, breaker panel, outdoor disconnect
- **Ice on unit**: Turn system OFF, run fan only. Call technician — likely low refrigerant or airflow restriction
- **Water leaking**: Check drain line for blockage. Place towels and call for service
- **Strange noises**: Turn off and call immediately — running with mechanical issues causes more damage
